# citas

API y utilidades para gestionar citas y estudiantes.

## Contenido
- `server.js` y `api_citas.js`: endpoints/API en Node.js.
- `fix_database.sql`: script SQL de apoyo para base de datos.
- `a.js` y `e.js`: pruebas/conexiones a base de datos Oracle.
- `claselunes.html`: recurso web de ejemplo.
- `pdf.py` y `distribuciones.py`: scripts auxiliares en Python.

## Uso rápido
1. Instalar dependencias: `npm install`
2. Ejecutar servidor: `node server.js`

## Nota
Este repositorio contiene solo la carpeta `Bases de datos` separada como proyecto independiente.
